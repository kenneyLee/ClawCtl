## 二、总体测试策略

### 2.1 关键决策

| 决策 | 理由 |
|------|------|
| **CommandExecutor 必须有完整的 mock 层** | LocalExec 用 `child_process.exec` 的 mock (vi.mock)，SshExec 用 mock ssh2 Client。绝不在 CI 中执行真实命令 |
| **配置写回需要 snapshot + rollback 测试** | 写坏配置 = 实例不可用，必须测：写前备份、写后验证、回滚恢复 |
| **服务控制 (start/stop/restart) 纯 mock** | 不在测试中真正杀进程，mock `child_process`/`ssh2` 的 exec 返回值 |
| **SSE 流测试用 EventSource mock** | 前端用 `EventSource` 接收日志流，测试中 mock SSE 端点 |
| **拓扑图测试只验证数据 → 节点/边映射** | ReactFlow 渲染不做像素级验证，只测数据转换逻辑和节点/边数量 |
| **Prompt injection 检测测试** | 用固定的 test cases（已知恶意/良性 prompt），不测 LLM 输出质量，只测检测 pipeline 的调用链 |
| **Live 测试扩展** | `test:live` 新增远程命令执行测试（只在有环境变量且指定 `--live-dangerous` 时运行） |

## 八、高风险测试重点

### 8.1 必须 100% 覆盖的危险路径

| 路径 | 风险 | 测试要求 |
|------|------|----------|
| 命令注入防护 (`sanitize.ts`) | 远程代码执行 | 所有已知注入模式必须被拦截，fuzz 测试 |
| 配置写回 (`config-editor.ts`) | 写坏配置导致实例不可用 | 写前自动备份 + 写后 JSON 验证 + 回滚测试 |
| 服务 Stop/Kill (`service.ts`) | 误杀进程 | PID 验证、只杀 OpenClaw 进程、确认弹窗 |
| SSH 凭据传输 (`executor.ts`) | 凭据泄露 | 凭据不出现在日志/响应中，内存使用后清零 |
| 权限模板应用 (`templates.ts`) | 错误模板导致权限失控 | 应用前 diff 预览 + 确认 + 自动快照 |
| OAuth token 持久化 (`settings.ts`) | 保存设置覆盖 OAuth 凭据 | PUT handler 合并逻辑 100% 覆盖 |
| OAuth token 刷新 (`client.ts`) | refresh 失败导致 LLM 不可用 | 刷新失败回退旧 token + onOAuthRefresh 回调 |
| Codex API 凭据传输 (`client.ts`) | token 泄露到日志 | Authorization header 仅在 fetch 中使用，不 console.log |
| JWT accountId 提取 (`client.ts`) | 解析失败导致 API 调用失败 | 畸形 token 抛明确错误 |

### 8.2 降级与容错

| 场景 | 预期行为 | 测试 |
|------|----------|------|
| SSH 连接断开中途 | 操作标记为 failed，不留残态 | mock 断连 |
| 配置写入到一半失败 | 自动回滚到 .bak | mock 写入 error |
| LLM 超时 | Digest/Injection 功能降级，不阻塞 | mock 30s timeout |
| 目标机器磁盘满 | 安装/升级提前检测，拒绝执行 | mock `df` 返回 0% 可用 |
| 并发操作冲突 | 同一实例不允许同时 restart + upgrade | 请求锁机制 |
| OAuth token 过期 | 自动刷新，刷新失败回退旧 token | mock expired + mock refresh fail |
| Codex API 返回非 200 | 抛明确错误含 status code | mock 400/429/500 |
| Codex SSE 流无完成事件 | 抛 "No response.completed event" | mock 不完整 SSE |
| OAuth 端口 1455 被占用 | startOpenAIOAuth 抛 "Cannot bind port" | mock EADDRINUSE |
| AI Assistant LLM 调用失败 | 返回 500 + 错误消息，不影响其他功能 | mock LLM chat throw |
| 模型列表 API 查询失败 | 降级返回静态模型列表 | mock fetch error |
